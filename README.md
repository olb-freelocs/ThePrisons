# ThePrisons

Client-side Fabric mod for Minecraft `1.21.11`.

## Links

- [Modrinth](https://modrinth.com/mod/theprisons)
- [Source](https://github.com/olb-freelocs/ThePrisons)

## Features

- Persistent pet cooldown cache across relogs and restarts
- Trinkets cooldown display in the same HUD
- Clean HUD for every tracked pet and trinket
- Centered ready announcements above the crosshair
- Pet names without level text
- `Ready` status styling with clear color separation
- Mod Menu integration with a modern config screen

## Build

```bash
./gradlew build
```
