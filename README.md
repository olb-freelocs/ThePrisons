# [PetReady](https://modrinth.com/mod/petready)

Client-side Fabric mod for Minecraft `1.21.11`.

## Links

- [Modrinth](https://modrinth.com/mod/petready)
- [Source](https://github.com/olb-freelocs/PetReady)

## Features

- Persistent pet cooldown cache across relogs and restarts
- Clean HUD for every tracked pet in the inventory
- Centered ready announcements above the crosshair
- Pet names without level text
- `Pet Ready` status styling with clear color separation
- Mod Menu integration with a modern config screen

## Build

```bash
./gradlew build
```
